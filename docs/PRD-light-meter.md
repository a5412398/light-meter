## 1. 产品概述

### 1.1 产品定位

Light Meter 是一款基于智能手机前置摄像头的专业级光线感应工具，旨在将复杂的农业/园艺光学指标平民化，帮助用户精准控制植物生长环境。

### 1.2 目标用户

| 用户群体 | 使用场景 | 核心诉求 |
|---------|---------|---------|
| 农业从业者 | 温室大棚、农田监测 | 低成本光照监测方案 |
| 园艺爱好者 | 家庭花园、室内植物 | 简单易用的光照管理 |
| 植物研究者 | 实验室环境控制 | 便携式数据记录 |

### 1.3 核心价值

- **零额外硬件**：利用现有手机摄像头实现光照监测
- **专业指标平民化**：Lux + CCT 双核心参数，覆盖 90% 植物光照需求
- **智能校准**：支持简单调节和专业参照校准，适配不同设备

| 功能模块 | 功能点 | 优先级 | 描述 |

### 2.2 功能依赖关系

```
F01 (实时监测)
  ├── 依赖 F07 (权限管理)
  └── 依赖 F02, F03 (计算模块)

F04 (校准功能)
  └── 影响 F02, F03 (计算精度)

F05 (历史记录)
  └── 依赖 F01, F02, F03 (数据来源)
```

## 3. 非功能需求

### 3.1 性能要求

| 指标 | 要求 | 说明 |
|------|------|------|
| 响应速度 | 参数更新延迟 < 500ms | 从摄像头捕获到 UI 更新 |
| 帧率 | ≥ 15 FPS | 预览画面流畅度 |
| 启动时间 | < 3 秒 | 冷启动到主界面可交互 |
| 内存占用 | < 150 MB | 运行时内存峰值 |

### 3.2 精度要求

| 参数 | 测量范围 | 精度目标 |
|------|---------|---------|
| Lux | 10 - 100,000 lx | ±15%（校准后） |
| CCT | 2,000 - 10,000 K | ±200 K |

### 3.3 稳定性要求

| 场景 | 要求 |
|------|------|
| 连续运行 | 30 分钟无崩溃 |
| 后台切换 | 恢复前台后正常工作 |
| 低电量 | 电量 < 10% 时正常工作 |
| 异常处理 | 摄像头被占用时显示友好提示 |

### 3.4 兼容性要求

| 项目 | 要求 |
|------|------|
| Android 版本 | Android 6.0 (API 23) 及以上 |
| 屏幕尺寸 | 适配 4.7" - 7" 屏幕 |
| 屏幕方向 | 支持竖屏模式 |
| 摄像头 | 需前置摄像头 |

## 4. 用户流程

### 4.1 首次使用流程

```
启动 App
  │
  ▼
检查摄像头权限
  │
  ├─ 未授权 → 弹出权限申请
  │              │
  │              ├─ 用户同意 → 进入主界面
  │              │
  │              └─ 用户拒绝 → 显示权限说明页
  │                              │
  │                              └─ 引导去设置开启
  │
  └─ 已授权 → 进入主界面
```

### 4.2 主界面流程

```
主界面
  │
  ├── 实时预览（摄像头画面 + 参数叠加）
  │
  ├── 点击 Lux 卡片 → 展开详情面板
  │
  ├── 点击 CCT 卡片 → 展开详情面板
  │
  ├── 点击校准系数 → 弹出校准面板
  │                    │
  │                    ├── 简单校准（滑块）
  │                    └── 参照校准（输入读数）
  │
  ├── 点击历史记录 → 跳转历史列表页
  │
  └── 点击设置 → 跳转设置页
```

### 4.3 校准流程

```
点击校准系数
  │
  ▼
校准面板弹出
  │
  ├── 方式一：简单校准
  │     │
  │     └─ 拖动滑块调整系数 (0.5x - 2.0x)
  │           │
  │           └─ 实时更新显示值
  │
  └── 方式二：参照校准
        │
        ├─ 输入专业照度计读数
        │
        ├─ 点击"计算校准系数"
        │
        └─ 系统自动计算并应用系数
```

## 6. 数据需求

### 6.1 数据模型

#### Measurement（测量记录）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | INTEGER | 是 | 主键，自增 |
| lux | REAL | 是 | 勒克斯值 |
| cct | REAL | 是 | 色温值（K） |
| calibration_factor | REAL | 否 | 当时使用的校准系数 |
| device_model | TEXT | 否 | 设备型号 |
| created_at | INTEGER | 是 | 时间戳（毫秒） |
| note | TEXT | 否 | 用户备注 |

#### CalibrationConfig（校准配置）

| 字段 | 类型 | 必填 | 说明 |
|------|------|------|------|
| id | INTEGER | 是 | 主键，自增 |
| name | TEXT | 是 | 配置名称 |
| factor | REAL | 是 | 校准系数 |
| reference_lux | REAL | 否 | 参照照度计读数 |
| is_default | INTEGER | 是 | 是否默认（0/1） |
| created_at | INTEGER | 是 | 创建时间戳 |

### 6.2 数据存储

- 存储方式：本地 SQLite 数据库
- 数据库位置：应用沙箱目录
- 数据库版本：v1

### 6.3 数据保留策略

| 策略 | 默认值 | 用户可配置 |
|------|--------|-----------|
| 保留天数 | 30 天 | 是（7/30/90/永久） |
| 自动清理 | 启动时检查 | 可关闭 |
| 最大记录数 | 无限制 | - |

## 8. 风险与对策

| 风险 | 影响 | 概率 | 对策 |
|------|------|------|------|
| 摄像头硬件差异大 | 精度不一致 | 高 | 提供校准功能 + 设备指纹库（二期） |
| 低光环境噪点多 | 读数波动大 | 中 | 多帧平滑 + 异常值过滤 |
| 后台运行限制 | 无法持续监测 | 中 | 提示用户保持前台运行 |
| 用户不理解校准 | 使用体验差 | 中 | 提供引导教程 + 默认配置 |
