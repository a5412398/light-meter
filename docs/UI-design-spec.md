# Light Meter - UI 设计规范

## 设计理念

**主题：自然有机风格**

将科技感与自然感融合，为农业/园艺场景打造专业但不冰冷的视觉体验。

## 动效规范

### 原则

- **自然感**：模拟植物生长的曲线（ease-out）
- **呼吸感**：微妙的脉冲动画
- **流畅性**：避免突兀的跳变

### 动画参数

| 动画 | 时长 | 缓动函数 | 用途 |
|------|------|---------|------|
| fadeIn | 200ms | ease-out | 元素出现 |
| slideUp | 300ms | ease-out | 弹窗出现 |
| pulse | 2000ms | ease-in-out | 数值呼吸 |
| ripple | 400ms | ease-out | 按钮点击 |

### 数值更新动画

```dart
// 数值平滑过渡
AnimatedNumber(
  value: lux,
  duration: Duration(milliseconds: 300),
  curve: Curves.easeOut,
)
```

## 页面布局

### 主界面

```
┌─────────────────────────────────────┐
│  Light Meter              ⚙️ 设置   │  ← AppBar (透明背景)
├─────────────────────────────────────┤
│                                     │
│     ┌─────────────────────────┐     │
│     │                         │     │
│     │   摄像头实时预览画面     │     │  ← CameraPreview
│     │                         │     │     (带圆角遮罩)
│     │   (全屏或 4:3 比例)     │     │
│     │                         │     │
│     └─────────────────────────┘     │
│                                     │
│     ┌───────────────────────────┐   │
│     │  🌞 1,234 Lux  [理想]    │   │  ← 参数叠加层
│     │  🌡️ 5,600 K   [日光]     │   │     (半透明背景)
│     └───────────────────────────┘   │
│                                     │
├─────────────────────────────────────┤
│                                     │
│  ┌─────────────┐  ┌─────────────┐   │
│  │  🌞 亮度    │  │  🌡️ 色温   │   │  ← 参数卡片
│  │  1,234 Lux │  │  5,600 K   │   │
│  │  [理想]    │  │  [日光]    │   │
│  └─────────────┘  └─────────────┘   │
│                                     │
│     📊 校准系数: 1.05x              │  ← 校准指示器
│     📅 历史记录 (3)                 │  ← 历史入口
│                                     │
└─────────────────────────────────────┘
```

### 设计要点

1. **摄像头预览**：
   - 占据屏幕上半部分
   - 圆角遮罩（底部圆角）
   - 参数叠加层使用半透明背景

2. **参数卡片**：
   - 左右并排布局
   - 点击展开详情
   - 状态颜色动态变化

3. **底部区域**：
   - 校准系数 + 历史入口
   - 底部安全区域留白

## 响应式设计

### 屏幕适配

| 屏幕尺寸 | 布局调整 |
|---------|---------|
| 小屏 (< 5") | 卡片垂直堆叠 |
| 中屏 (5-6") | 标准布局 |
| 大屏 (> 6") | 增大卡片尺寸 |

### 横竖屏

- **竖屏**：标准布局
- **横屏**：摄像头预览左，参数卡片右

## 无障碍设计

### 颜色对比度

- 主文字：对比度 ≥ 4.5:1
- 次要文字：对比度 ≥ 3:1
- 状态色：对比度 ≥ 3:1

### 触摸目标

- 最小触摸区域：48dp × 48dp
- 按钮间距：≥ 8dp

### 动效控制

- 提供"减少动效"选项
- 遵循系统无障碍设置
